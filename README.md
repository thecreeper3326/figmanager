This repository is for my FigManager project. It is a simple configuration manager with saving, loading, and some rich data types that also hold descriptions, names, minimum/maximum values, etc. 
One can download this project and publish it to their local maven repository, there, it will be able to load extensions to add more functionality.
Right now the only extension ive made for it is a Minecraft layer that adds a config command and a nice dynamic gui, but others can be built very easily, for things like JavaFX or maybe Swing.
